# Proyecto divide y venceras AED2

primer proyecto de la asignatura de algoritmos y estructura de datos 2 de la universidad de Murcia.

Para ejecutar el main del proyecto: 
```bash
make
```

para ejecutar los unit tests:
```bash
make tests
```

estructura del proyecto:

```
DyV-AEDII
 ┣ docs -> pseudo codigo de los archivos de cpp
 ┃ ┣ DyV.md
 ┃ ┗ Iterativo.md
 ┣ src -> codigo del proyecto
 ┃ ┣ DyV.cpp
 ┃ ┣ Iterativo.cpp
 ┃ ┣ Main.cpp
 ┃ ┗ Proyecto.hpp
 ┣ test -> unit testing
 ┃ ┗ testEj5.cpp
 ┣ vendor
 ┃ ┗ doctest -> submodulo de git de la libreria de unit testing
 ┣ .gitignore
 ┣ .gitmodules
 ┣ Makefile
 ┗ readme.md
```
